The ethical hacker's map

Click any hotspot to learn about that vulnerability — what it is and what tools hackers use. Attackers can enter at any layer, not just through the user.

Attack surface layers
Human & Social engineering
Often the easiest entry point — no technical knowledge required. A convincing email or phone call can hand over credentials or grant direct access without touching a line of code. Automated credential stuffing is a separate web/application attack — see the server panel.
Network
The pipes connecting everything. Attackers scan for open ports, intercept unencrypted traffic, or exploit weak credentials on internet-facing devices like routers and cameras.
Hardware / CPU
The foundation everything runs on — present in both laptop and server. Exploits manipulate how the CPU executes code directly, at the level of registers and memory.
OS & Services
Sits on top of the server hardware. After gaining initial access, attackers escalate privileges. Misconfigured files and unpatched services are the most common paths upward.
Local software
Applications running on the laptop. Bugs in local software can be exploited to execute arbitrary code or escalate privileges on that machine.
Web & Application
The app running on top of the server. Websites and APIs are often the most exposed surface — attackers inject commands, steal session tokens, or manipulate URLs to access other users' data.
